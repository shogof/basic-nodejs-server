Welcome to My Node.js Server

This server is built using Node.js and Express.js.

Project Features

  • Static Homepage:
    • Serves a static HTML file as the homepage.
  • CRUD Operations:
    • Manage Users and Products:
    • GET /users - Retrieve all users.
    • POST /users - Create a new user.
    • PUT /users/:id - Update user details.
    • DELETE /users/:id - Delete a user.
    • GET /products - Retrieve all products.
    • POST /products - Create a new product.
    • PUT /products/:id - Update product details.
    • DELETE /products/:id - Delete a product.
  • Static Assets:
    • Serves CSS and JavaScript files from the public directory.
  • Data Handling:
    • Handles JSON data in request bodies for POST and PUT requests.
  • Routing:
    • Organizes routes using Express Router.
